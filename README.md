# Human Hustle – Rise in the Corporate World 🎮

A corporate-themed Snake and Ladders game built using HTML, CSS, and JavaScript.

## Features
- Career-based Snake & Ladders board (Fresh Graduate → CEO)
- AI mode gameplay
- Quiz-based promotion system
- Animated dice & smooth movement
- Corporate UI with Tailwind CSS

## Tech Stack
- HTML
- CSS (Tailwind)
- JavaScript

## How to Run
Open `human-hustle.html` in any browser.
